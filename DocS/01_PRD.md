# OfficeFlow ERP Backend PRD

## Objective
Create a production-ready backend API for OfficeFlow ERP frontend.

Frontend:
React + TypeScript + TanStack Router

Backend goal:
Replace all mock/local storage data with PostgreSQL database APIs.

## Priority Phase

PHASE 1:
- Authentication
- Employee Management
- Employee Hierarchy
- Access Control
- Task Management

PHASE 2:
- Attendance
- Leave
- Payroll
- Reports

PHASE 3:
- WMS
- Hiring

## Core Features

Authentication:
JWT login system.

Employee:
- Add employee
- Update employee
- Disable employee
- Upload documents
- Store hierarchy

Tasks:
- Create task
- Assign task
- Update task
- Maintain task history

Profile:
- Attendance
- Leave history
- Salary progression graph

All demo frontend data must be migrated into PostgreSQL seed data.
