# Role Permission Matrix


## SUPER ADMIN

Allowed:
- All modules
- Create employee
- Update employee
- Assign permissions


## Department Head

Modules:

1 Employee Management
2 Task Management


Permissions:

- View department employees
- Assign tasks
- Update task status


## Payroll Manager

Modules:

1 Employee
2 Payroll


Permissions:

- Salary update
- Payroll request
- Salary history


## Employee

Single Module:

Employee Portal


Allowed:

- View profile
- Attendance
- Leave request
- Assigned tasks only


## Others

Maximum 4 assigned modules using RBAC.
